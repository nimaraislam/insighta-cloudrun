Step 9 — Bonus (CI/CD)

Create folder:

.github/workflows

Create file:

deploy.yml

Add:

name: Deploy to Cloud Run

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout
      uses: actions/checkout@v3

    - name: Google Cloud Auth
      uses: google-github-actions/auth@v1
      with:
        credentials_json: ${{ secrets.GCP_SA_KEY }}

    - name: Deploy
      uses: google-github-actions/deploy-cloudrun@v1
      with:
        service: insighta-app
        image: gcr.io/YOUR_PROJECT_ID/insighta-app
        region: europe-west1

Step 9 — Implement CI/CD with GitHub Actions
1️⃣ Create a Google Cloud Service Account

Go to:

Google Cloud Console → IAM & Admin → Service Accounts

Click Create Service Account

Name:

github-actions-deployer

Click Create and Continue

2️⃣ Give Required Permissions

Add these roles:

Cloud Run Admin

Storage Admin

Service Account User

Cloud Build Editor

Click Done

These permissions allow GitHub to:

build containers

push images

deploy to Cloud Run

3️⃣ Generate a JSON Key

Open the service account.

Go to Keys → Add Key → Create new key

Select:

JSON

Download the file.

Example:

github-actions-key.json
4️⃣ Add the Key to GitHub Secrets

Open your GitHub repository:

https://github.com/YOUR_USERNAME/insighta-cloudrun

Then go to:

Settings → Secrets and variables → Actions

Click:

New repository secret

Name:

GCP_SA_KEY

Paste the entire JSON file content.

Save.

5️⃣ Create the GitHub Actions Workflow

Inside your project folder create:

.github/workflows

Then create a file:

deploy.yml

Your project should look like:

insighta-cloudrun
│
├ app.py
├ Dockerfile
├ requirements.txt
└ .github
    └ workflows
        └ deploy.yml
6️⃣ Add the CI/CD Workflow Code

Replace YOUR_PROJECT_ID with your real Google Cloud project ID.

name: Deploy to Cloud Run

on:
  push:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout repository
      uses: actions/checkout@v3

    - name: Authenticate to Google Cloud
      uses: google-github-actions/auth@v1
      with:
        credentials_json: '${{ secrets.GCP_SA_KEY }}'

    - name: Set up gcloud
      uses: google-github-actions/setup-gcloud@v1

    - name: Build Docker Image
      run: |
        gcloud builds submit --tag gcr.io/YOUR_PROJECT_ID/insighta-app

    - name: Deploy to Cloud Run
      run: |
        gcloud run deploy insighta-app \
        --image gcr.io/YOUR_PROJECT_ID/insighta-app \
        --platform managed \
        --region europe-west1 \
        --allow-unauthenticated
